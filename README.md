## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| 🤝 **Squad-Split** | Built-in Splitwise logic with UPI deep-linking for instant settlement. |
| 📊 **Neon Analytics** | Heatmaps and contribution graphs that track your savings streaks. |
| 📱 **UPI Deep-Sync** | Auto-categorization of spends (Zomato 🍜, Amazon 👟) via SMS/Statement parsing. |
| 🕵️ **Privacy First** | Local-first storage. Your data stays on your device. No bank logins required. |
